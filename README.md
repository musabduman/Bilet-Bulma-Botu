🚆 TCDD Bilet Avcısı Botu
TCDD'nin hızla tükenen YHT ve anahat tren biletlerini yakalamayı otomatize eden akıllı bir Telegram asistanı! Bilet bulmak için sürekli sistemi yenileme devri bitti; kriterlerinizi belirleyin, bot arkanızda 7/24 koltuk kovalasın.

✨ Öne Çıkan Özellikler:

Kesintisiz Otomasyon: Belirlediğiniz rota ve saat aralığını arka planda düzenli olarak tarar.

Akıllı Koltuk Analizi (Cinsiyet Bazlı Filtreleme): TCDD'nin vagonlardaki kadın/erkek yan yana oturma kurallarını API üzerinden okuyup hesaplayarak, yalnızca doğrudan bilet alabileceğiniz geçerli koltukları size sunar.

Anında Bildirim: Kriterlerinize uygun boş bir yer açıldığı saniye, direkt bilet alma bağlantısıyla birlikte Telegram üzerinden uyarı gönderir.

Dinamik İstasyon Yönetimi: Web API ile senkronize çalışarak en güncel TCDD istasyon verilerini kullanır.
